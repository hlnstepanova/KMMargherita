# KMMargherita

A simple project that shows how to share Networking and Viewmodel layers between iOS and Android, using: <br>
* Ktor
* Koin
* Jetpack Compose
* SwiftUI

### What you need:
* Android Studio
* XCode + Tools
* RubyGems

A spin-off from [KMPizza](https://github.com/hlnstepanova/kmpizza-repo)
